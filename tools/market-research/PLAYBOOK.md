# Playbook: автоматизация разведки офферов в Европе

Методология: как автоматизировать поиск лучших офферов в одной стране Европы
и сравнивать выдачу Facebook-рекламы между странами, чтобы видеть **спрос** и
**конкуренцию**. Скрипты в этой папке автоматизируют шаги 1–2; здесь — полная
картина и no-code-альтернативы.

## Воронка из 3 слоёв

| Слой | Вопрос | Инструмент | Автоматизировано здесь |
|------|--------|-----------|:---:|
| **1. Спрос** | Что ищут/покупают в нише | Google Trends, Etsy, TikTok Creative Center | ✅ Trends (`collect.py`) |
| **2. Конкуренция** | Кто уже льёт рекламу и на что | Facebook Ad Library API | ✅ (`collect.py` + токен) |
| **3. Оффер** | Какая связка товар+креатив+цена выигрывает | долгоживущие объявления, spy-инструменты | частично |

## Ключевая идея сравнения стран

Одно и то же объявление, которое **крутится долго в стране A, но которого нет
в стране B** — это либо неохваченный спрос (возможность зайти), либо сигнал,
что там оно не работает. **Долгожительство объявления** (`median_ad_age_days`)
= косвенный признак прибыльного оффера: убыточную рекламу быстро выключают.

Матрица решений товар × страна:

| Спрос | Конкуренция | Вывод |
|-------|-------------|-------|
| высокий | низкая | 🟢 приоритет запуска — свободный спрос |
| высокий | высокая | 🟡 есть рынок, но нужен сильный оффер/УТП |
| низкий | низкая | ⚪ ниша/тест, осторожно |
| низкий | высокая | 🔴 пропустить |

`opportunity_score` в выгрузке = формализация этой матрицы (спрос ÷ конкуренция).

## Пошаговый процесс (шаг 1 — собрать лучшие товары)

1. **Список ключей на локальных языках** по каждой стране — в `keywords.json`.
   Без локальных ключей выдача нерелевантна.
2. **Спрос** — `python collect.py --no-fb` (Google Trends, без токена).
3. **Конкуренция** — добавить `META_ACCESS_TOKEN`, запустить `python collect.py`.
4. **Ранжирование** — открыть `out/market_research.csv`, отсортировано по
   `opportunity_score`. Верх списка = кандидаты в «лучшие товары».
5. **Ручная валидация топ-5** — открыть `ad_snapshot_url` победителей, посмотреть
   креативы и цены живых объявлений.

## Расширение (шаг 3 — офферы и креативы)

Что добавить к скриптам по мере роста:

- **Etsy Open API** — бестселлеры и цены по нише (для этники Etsy в ЕС ключевой).
  Даёт реальные товары-лидеры, а не только рекламу.
- **TikTok Creative Center** (бесплатно) — топ-объявления по странам; для молодой
  аудитории часто опережает FB.
- **Google Keyword Planner / Ahrefs / Semrush** — абсолютные объёмы поиска
  (Trends даёт только относительный индекс 0–100).

## No-code альтернативы (если не хотите поддерживать код)

Платные spy-сервисы с фильтром по стране/языку/долгожительству объявления:

- **Minea** — FB + TikTok, заточен под e-commerce/дропшиппинг-офферы.
- **BigSpy**, **AdSpy** — большие базы FB/Instagram-рекламы.
- **Dropispy** — фокус на FB, доступный по цене.
- **PiPiADS** — для TikTok.

Они закрывают шаги 2–3 без разработки, но дороже и не дают своей единой таблицы
товар × страна — её удобнее строить скриптом.

## Автоматизация по расписанию

Чтобы отслеживать динамику, запускайте `collect.py` регулярно (напр. еженедельно)
через cron / GitHub Actions и складывайте выгрузки с датой — так видно тренд
спроса и рост/спад конкуренции по каждому товару и стране во времени.

## Юридическое / этичное

- Facebook Ad Library API — **официальный** и легальный источник (в ЕС открыт
  по DSA). Используйте его, а не парсинг закрытых страниц.
- Соблюдайте rate limits Trends и Meta, не обходите капчи и ToS площадок.
