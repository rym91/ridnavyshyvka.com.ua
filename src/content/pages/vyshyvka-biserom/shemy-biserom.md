Схема — це основа будь-якої роботи бісером. Саме від того, наскільки правильно вона побудована й наскільки зрозуміло читається, залежить результат: рівні ряди, чисті переходи кольору та точний малюнок. На цій сторінці зібрано все, що варто знати про схеми для вишивки бісером — їх види, формати та принципи читання.

Якщо ви тільки знайомитеся з технікою, почніть з простих сюжетів і поступово ускладнюйте завдання. Загальний контекст про напрямок дає [розділ про вишивку бісером](/vyshyvka-biserom), а суміжні техніки можна порівняти на сторінці [технік вишивки](/tehniky-vyshyvky).

## Які бувають схеми вишивки бісером

Залежно від способу нанесення бісеру та типу виробу, схема вишивки бісером може бути:

- **клітинкова (кольорова карта)** — найпоширеніший варіант, де кожна клітинка відповідає одній бісерині певного кольору;
- **символьна** — поряд із кольором використовуються значки, що зручно при друці чорно-білих схем;
- **схеми для повного зашиття** — увесь фон заповнюється бісером, малюнок не має «порожніх» зон;
- **схеми часткового зашиття** — бісером виділяють лише центральний сюжет, фон залишається тканинним.

Окремий клас — схеми для плетіння (гердани, браслети, силянки), які читаються по-іншому, ніж карти для пришивання на канву.

## Як читати схему бісером

Принцип нагадує роботу з [схемами для вишивки хрестиком](/vyshyvka-hrestykom/shemy-hrestykom): полотно схеми ділиться на клітинки, кожна з яких — це одна бісерина. До схеми додають ключ (легенду) з відповідністю кольорів і номерів бісеру.

Базові кроки читання:

1. Знайдіть центр схеми — зазвичай він позначений стрілками по краях.
2. Звіртеся з ключем: який номер бісеру відповідає якому кольору.
3. Визначте напрямок шиття — найчастіше зліва направо й знизу вгору.
4. Рахуйте бісерини по рядах, орієнтуючись на зміну кольору.

## Схеми з бісеру для початківців

Для перших робіт обирайте невеликі схеми з обмеженою палітрою — 4–8 кольорів. Прості геометричні мотиви, квіти чи окремі елементи орнаменту дозволяють відпрацювати рівність ряду без складних переходів. Корисні поради для старту зібрано на сторінці [вишивка для початківців](/ukrayinska-vyshyvka/vyshyvka-dlya-pochatkivciv).

Ознаки вдалої схеми для новачка:

- невелика площа зашиття;
- чіткий контраст між кольорами;
- мінімум дрібних деталей в один-два стібки.

## Матеріали під схему

Підбір бісеру та основи безпосередньо пов'язаний зі схемою: розмір бісерин впливає на кінцеві пропорції малюнка. Для рівномірного полотна важлива якісна основа — про варіанти читайте на сторінці [канва для вишивки](/materialy-dlya-vyshyvky/kanva-dlya-vyshyvky). Що складніша схема, то якіснішим має бути бісер, аби кольори не «грали» в готовому виробі.

Готові до конкретних сюжетів? Перегляньте схеми [герданів](/vyshyvka-biserom/herdany-shemy) та [ікон бісером](/vyshyvka-biserom/vyshyvka-biserom-ikony) — для них діють свої правила побудови.
